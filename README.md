# St Johns Hospital - Page Overview

This document explains the purpose of each page linked in the website navigation bar.

## Home (index.html)
The main landing page of the website.  
Provides a general overview of the hospital, highlights key services, and guides visitors to important sections.

## About (about.html)
Shares information about the hospital, including:
- Mission and vision
- History
- Core values
- Management or leadership overview

## Services (services.html)
Displays the medical services offered, such as:
- General Medicine
- Emergency Care
- Surgery
- Laboratory Services
- Specialized treatments

## Appointment (appointment.html)
Enables patients to:
- Book appointments
- Select doctors or departments

## Blog (blog.html)
Provides:
- Health tips
- Medical news
- Awareness articles
- Hospital updates

Keeps patients informed and engaged.

## Contact Us (contact.html)
Allows visitors to:
- View hospital contact details
- Send inquiries through a contact form
- Find location information
